CAN 통신 프로젝트

Arduino / ESP8266과 MCP2515 CAN 모듈을 활용하여 여러 노드 간 CAN 통신을 실습한 프로젝트입니다.
CAN 메시지를 이용하여 LED를 번갈아 제어하고, 적외선 근접센서 입력에 따라 다른 노드의 LED를 제어하며, 먼지센서와 온습도센서 데이터를 LCD에 표시하는 기능을 구현했습니다.

프로젝트 개요
CAN 마스터 노드
        │
        │ CAN 통신
        ▼
MCP2515 CAN Bus
        │
        ├── 슬레이브1 : LED 제어
        │
        ├── 슬레이브2 : LED 제어 / 적외선 근접센서 입력
        │
        └── 센서 노드 : 먼지센서, 온습도센서 데이터 측정
                    │
                    ▼
                LCD 표시
프로젝트 구성
#1 CAN 통신으로 LED 번갈아 깜빡이기

Arduino 또는 ESP8266 보드에 MCP2515 CAN 모듈을 연결하여 CAN 마스터와 슬레이브 노드를 구성했습니다.
마스터는 서로 다른 CAN ID를 가진 슬레이브1, 슬레이브2에 제어 메시지를 전송하고, 각 슬레이브는 수신한 데이터의 첫 번째 바이트 값에 따라 LED를 ON/OFF 합니다.

마스터 노드
   │
   │ CAN Frame 전송
   ▼
CAN Bus
   │
   ├── 슬레이브1 ID : 0x0F6 → LED 제어
   └── 슬레이브2 ID : 0x036 → LED 제어
동작 방식
마스터 → 슬레이브1 : ID 0x0F6, DATA[0] = 0x01 → 슬레이브1 LED ON
마스터 → 슬레이브2 : ID 0x036, DATA[0] = 0x00 → 슬레이브2 LED OFF

1초 후

마스터 → 슬레이브1 : ID 0x0F6, DATA[0] = 0x00 → 슬레이브1 LED OFF
마스터 → 슬레이브2 : ID 0x036, DATA[0] = 0x01 → 슬레이브2 LED ON

이를 통해 두 개의 슬레이브 LED가 CAN 메시지에 따라 번갈아 깜빡이도록 구현했습니다.

#2 적외선 근접센서를 이용한 LED 제어

슬레이브1에는 적외선 근접센서를 연결하고, 슬레이브2에는 LED를 연결했습니다.
슬레이브1이 센서 입력을 감지하면 CAN 메시지를 마스터에게 전송하고, 마스터는 해당 정보를 바탕으로 슬레이브2에 LED 제어 명령을 전달합니다.

슬레이브1
적외선 근접센서 감지
        │
        │ CAN 메시지 전송
        ▼
마스터 노드
        │
        │ LED 제어 명령 전달
        ▼
슬레이브2
LED ON / OFF
CAN 메시지 구조
[CAN ID] [작업 내용] [명령 값] [나머지 빈 데이터]
메시지 예시
슬레이브1 → 마스터
ID : 0x0F6
DATA[0] : 0x02
DATA[1] : 0x01
의미 : 적외선 근접센서에서 물체 감지

마스터 → 슬레이브2
ID : 0x036
DATA[0] : 0x01
DATA[1] : 0x01
의미 : 슬레이브2 LED ON

마스터 → 슬레이브2
ID : 0x036
DATA[0] : 0x01
DATA[1] : 0x00
의미 : 슬레이브2 LED OFF
동작 과정
1. 슬레이브1의 적외선 근접센서가 물체를 감지한다.
2. 슬레이브1은 CAN 메시지로 감지 상태를 마스터에게 전송한다.
3. 마스터는 수신한 메시지를 분석한다.
4. 마스터는 슬레이브2에게 LED ON 명령을 전송한다.
5. 슬레이브2는 CAN 메시지를 수신하고 LED를 켠다.
6. 물체가 사라지면 OFF 명령을 전송하여 LED를 끈다.
#3 먼지센서 및 온습도센서 데이터 LCD 표시

센서 노드에는 먼지센서와 온습도센서를 연결하여 환경 데이터를 측정했습니다.
측정된 센서값은 CAN 메시지로 마스터 또는 표시 노드에 전달되며, 수신 노드는 데이터를 해석하여 LCD에 출력합니다.

센서 노드
먼지센서 + 온습도센서
        │
        │ CAN 데이터 전송
        ▼
CAN Bus
        │
        ▼
LCD 표시 노드
온도 / 습도 / 먼지 농도 표시
센서 데이터 예시
온도 : 24.50 ℃
습도 : 55.20 %
먼지 농도 : 120
CAN 데이터 구성 예시
DATA[0] : 작업 코드
DATA[1] : 온도 상위 바이트
DATA[2] : 온도 하위 바이트
DATA[3] : 습도 상위 바이트
DATA[4] : 습도 하위 바이트
DATA[5] : 먼지센서 상위 바이트
DATA[6] : 먼지센서 하위 바이트
DATA[7] : 예비 데이터

온도와 습도는 소수점 처리를 위해 정수형으로 변환하여 전송했습니다.

예시 : 24.50 ℃ → 2450으로 변환
상위 바이트 : 2450 / 256
하위 바이트 : 2450 % 256

수신 측에서는 다시 두 바이트를 결합한 뒤 100으로 나누어 실제 온도값으로 복원합니다.

수신값 = 상위바이트 × 256 + 하위바이트
실제값 = 수신값 / 100.0
